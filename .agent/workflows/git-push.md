1. Check `git status` to see the current changes.
// turbo
2. Run `git add .` to stage all changes.
3. Review the staged changes and generate a concise, descriptive commit message in English.
4. Run `git commit -m "<message>"` with the generated message.
// turbo
5. Run `git push` to push the changes to the remote repository.
