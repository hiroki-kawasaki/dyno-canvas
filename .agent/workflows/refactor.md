---
description: Rrefactor Workflow
---

1. Understand the specified refactoring.
2. Perform refactoring.
3. Once development is complete, run the lint check: `npm run lint`.
4. Make corrections based on the lint check results.
5. After passing the lint check, run the tests: `npm run test`.
6. Make necessary corrections based on the test results.
7. Ensure the lint check, tests, and build (`npm run build`) complete without errors.
